# 大连交通大学有线网自动认证工具 (DJTU Network Configurator)

本项目专为大连交通大学（DJTU）的同学们开发，旨在彻底替代之前繁琐的有线校园网认证流程。只需一次配置，即可实现“开机即上网”的无感顺滑体验。

> **💡 声明**：本软件目的是方便同学上网，完全免费。项目核心代码大部分由 AI 辅助编写完成。

---

## 核心功能
* 一键网络配置：自动检测活跃有线网卡，一键写入 IPv4、子网掩码、网关与 DNS，告别每次重装系统或换网口后手动配 IP 的折磨。
* 锐捷全自动登录：内置双引擎自动化（底层 UI 控件注入 + 备用键盘模拟），全自动输入学号密码并完成锐捷客户端认证。
* 极致无感体验：支持“开机静默启动”与“启动后自动执行认证”，运行后自动隐藏至系统托盘，不打扰任何日常操作。
* 自动更新检测：对接 GitHub Releases 接口，一键检查并在线下载最新版本，时刻保持最佳兼容性。

---

## 下载与安装

1.  前往本项目的 [Releases 页面](https://github.com/Crazy-chaos/DJTUxiaoyuanwang/releases)下载最新版本的 `.exe` 可执行文件。
2.  将软件放置在一个固定的文件夹中（注意：不要放在系统受限目录或随时会清理的临时文件夹中，以免影响开机自启和配置保存）。
3.  双击运行程序。由于修改网卡 IP 和设置开机自启任务需要最高权限，系统会弹出 UAC 提示，请点击“**是 / 允许**”。

---

## 使用指南

### 第一步：配置网络参数
* 将你的电脑通过网线连接到宿舍网口。
* 在软件“第一步”区域填入你的专属 IP 地址、网关等信息。
    > **不知道填什么？** > 鼠标悬停在“不知道填什么？”提示上，通过微信扫码进入“宿舍 IP 查询”页面获取。

### 第二步：配置锐捷认证
* **锐捷账号**：填入你的本人一卡通号（即学号，如 `2612010110`）。
* **锐捷密码**：填入本人生日（8位数字，如 `19940101`）。
* 勾选“执行此步骤以自动连接锐捷认证”。

### 第三步：保存并享用
* 点击 **“应用配置 (需管理员)”** 按钮。
* （强烈建议）点击右上角 **“⚙️ 设置”**，勾选 **“开机静默启动”** 与 **“开机/启动后自动执行网络配置与锐捷认证”**。
* 大功告成！以后每次开机，它都会在后台默默为你铺好上网之路。

## 🙋‍♂️ 常见问题与反馈

**Q: 软件运行时报错或者无法修改 IP 怎么办？**
A: 请确保你使用的是电脑的主板原生网口或靠谱的扩展坞，并且在软件启动时授予了管理员权限。如果提示“未找到网线”，请检查网线是否插紧或网口是否损坏（软件界面内有报修入口）。
